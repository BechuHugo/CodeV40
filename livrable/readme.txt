Dans le dossier livrable il faut créer les dossiers suivants :
- app: Applications testées et prêtes à l'emploi avec une description (généralement il s'agit d'un installateur ou d’un exécutable)
- bin: Binaire des développements (ci nécessaire)
- doc: Manuel et guide d'installation
- src: Code source des développements
- rapport: Différents rapports + présentations des exposés et soutenance (fichier source et au format pdf)
Et dans ce fichier readme.txt, toute information utile à la prise en main du livrable par le client comme :
Compte machine:
	login: utilisateur
	mp: password
